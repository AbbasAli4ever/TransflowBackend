# Roadmap

This roadmap defines phased delivery and keeps scope disciplined. V1 is focused on core posting and balances; later phases add returns, transfers, and SaaS features.

## V1 (Core)

- Purchase and Sale posting
- Supplier and Customer payments
- Inventory movements and ledger/payment entries
- Statements and balance queries
- Manual opening balances

## V1.1 (Quality)

- Supplier and Customer returns
- Internal transfers
- Voids and adjustments
- Basic export (CSV)

## V2 (Growth)

- Roles and permissions (RBAC)
- Import templates and batch rollback UI
- Multi-tenant SaaS
- Additional reports

## V3 (Scale)

- Multi-currency
- Multi-warehouse
- Advanced analytics
